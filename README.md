# LEDイルミネーション シミュレーター

建物から垂直に垂らすLEDテープの視覚効果を、実際の設置前にブラウザで確認できるWebツールです。

**デモURL**: https://led-illumination-simulator.vercel.app

![LEDシミュレーター](https://img.shields.io/badge/Status-Online-brightgreen) ![Version](https://img.shields.io/badge/Version-1.0.0-blue)

---

## 特徴

### 21種類のアニメーションパターン

| カテゴリ | パターン |
|---------|---------|
| **自然現象** | 滝、逆滝、雨、雪、流星、雷、炎 |
| **波・動き** | 波、スキャナー、チェイス、シアターチェイス、矢印（上/下） |
| **パターン** | ストライプ、ドット、グラデーション |
| **光の効果** | きらめき、トゥインクル、フラッシュ、鼓動 |
| **テキスト** | 文字スクロール表示 |

### カスタマイズ機能

- **高さ調整**: 1m〜10m（LEDテープの長さ）
- **本数調整**: 3本〜15本（並べるテープの数）
- **速度調整**: 10〜100（アニメーション速度）
- **カラー選択**: カラーピッカー + 6色のプリセット

### 対応デバイス

- PC（Windows / Mac / Linux）
- タブレット（iPad / Android）
- スマートフォン（iPhone / Android）

## 使い方

### 1. アクセス

ブラウザで以下のURLを開きます：
https://led-illumination-simulator.vercel.app

### 2. パターンを選択

「パターン選択」ドロップダウンから好きなアニメーションを選びます。

### 3. パラメータを調整

スライダーで高さ、テープ本数、速度を調整します。

### 4. カラーを選択

- カラーピッカーで自由に色を選択
- または6色のプリセットから選択

### 5. 再生

「▶️ 再生」ボタンを押すと、選択したパターンでLEDが動き始めます。

### 6. 停止・調整

「⏸️ 一時停止」ボタンで停止し、設定を調整できます。

## パターン紹介

### 💧 滝（waterfall）
連続的に大量の水が流れ落ちる表現。途切れることなく、まるで本物の滝のように見えます。

### ☄️ 流星（meteor）
ポツリポツリと流れ星が流れる表現。長い尾を引いて、夜空を流れる流星のようです。

### ❄️ 雪（snow）
ゆっくり落ちる雪の結晶。中心が明るく、周辺が少し光ります。

### 🌧️ 雨（rain）
複数の水滴が落ちる表現。雨が降る様子を再現しています。

### ⚡ 雷（lightning）
ランダムにフラッシュする雷の表現。予測不可能な激しい光。

### 🌊 波（wave）
サインカーブで上下に波打つ表現。海の波のようなリズム。

### ✨ きらめき（sparkle）
ランダムにLEDが点灯する、星のきらめきのような効果。

### 💓 鼓動（heartbeat）
ドクンドクンと脈打つ表現。心臓の鼓動のようなリズム。

### 📝 テキスト（text）
「LED」の文字が上から下にスクロールして表示されます。

## よくある質問

### Q. 実際のLED制御に使えますか？
A. このシミュレーターは視覚効果の確認用です。実際の制御には、Arduino/ESP32などのマイコンが必要です。

### Q. スマホで見ることはできますか？
A. はい、スマートフォンのブラウザでも動作します。URLをQRコードにして共有することもできます。

### Q. 設定を保存できますか？
A. 現在は保存機能はありませんが、今後のバージョンで実装予定です。

### Q. 自分で新しいパターンを作れますか？
A. このWebツールでは作成できませんが、開発者向けドキュメント（led_simulator_spec.md）を参照してカスタマイズ可能です。

### Q. オフラインで使えますか？
A. インターネット接続が必要です。オフライン版が必要な場合はご相談ください。
